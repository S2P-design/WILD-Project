So fügst du die Signatur ein:

  1. Markiere die adaptive Version oben (von „René Wild" bis „NOT BY DEFAULT.")
  2. Kopiere mit Cmd + C
  3. Apple Mail: Einstellungen → Signaturen → Neue Signatur → Einfügen mit Cmd + V
  4. Gmail: Einstellungen → Alle Einstellungen → Signatur → Einfügen
  5. Outlook: Einstellungen → E-Mail → Verfassen → Signatur → Einfügen

Die adaptive Version passt sich automatisch an Hell-/Dunkelmodus an (Apple Mail, iOS Mail, Outlook neu). In Gmail werden die Light-Mode-Farben verwendet.
